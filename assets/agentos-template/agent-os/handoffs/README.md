# Agent OS Handoffs

Handoff records for this Agent OS live in this directory.

Use this minimum shape:

```yaml
handoff:
  active_user_object:
  task_contract:
  current_route:
  completed_evidence:
  evidence_gaps:
  blockers:
  side_routes_parked:
  next_safe_action:
```

Do not write unverified claims into handoff as settled facts. Run Evidence-to-Claim Gate first.
